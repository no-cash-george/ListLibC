# ListLibC 📚

A lightweight, efficient implementation of a **Singly Linked List** written in pure C.

## 🚀 Overview
This library was developed to demonstrate low-level memory management and data structure implementation without relying on high-level abstractions. It provides a robust API for dynamic data storage using manual memory allocation.

## 🛠️ Technical Features
* [cite_start]**Manual Memory Management:** Full control over allocation and deallocation using `malloc` and `free` to prevent memory leaks. [cite: 8, 23]
* **Singly Linked List Logic:** Implementation of core operations:
    * Node creation and initialization.
    * Efficient insertion (head/tail/index).
    * Node deletion and list clearing.
    * Traversal and data retrieval.
* [cite_start]**Header-Only Potential:** Structured with clear separation of interface and implementation for easy integration. 

## 💻 Technologies Used
* [cite_start]**Language:** C (C99/C11 standard) 
* **Tools:** GCC / Clang compiler

## 📂 Project Structure
* [cite_start]`list.h`: Function prototypes and structure definitions. 
* [cite_start]`list.c`: Implementation logic. 
* [cite_start]`main.c`: Test suite and usage examples.
